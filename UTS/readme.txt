🔍 Ringkasan Model Lain:
DBSCAN:

Bisa deteksi outlier
– Sensitif terhadap parameter ε ➜ hasil tidak konsisten

Agglomerative Clustering:

Bisa ungkap struktur hierarki
– Cenderung overfit, apalagi jika cluster tidak seimbang

Spectral Clustering:

Cocok untuk bentuk cluster kompleks
– Komputasi berat & terlalu detail ➜ overfit pada data sederhana

GMM:

Bagus untuk cluster tumpang tindih
– Performa lebih rendah, hasil kurang jelas secara visual

✅ Kesimpulan Akhir:
KMeans adalah pilihan terbaik karena konsisten unggul di metrik evaluasi dan visualisasi, serta paling cocok untuk struktur data yang ada.
